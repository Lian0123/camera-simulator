# 變更紀錄

## [0.0.4] - 2026-01-31

### 新增功能
- ✨ **修復 RWD 手機滑動問題**
  - 實裝 iOS 光滑滾動 (`-webkit-overflow-scrolling: touch`)
  - 優化各斷點的佈局適配
  - 移動設備滾動體驗大幅提升
- ✨ **對焦距離設置（Focus Distance）**
  - 在 Lens & DOF 中新增對焦距離滑塊
  - 範圍：0.1m ~ 100m（符合實際拍攝需求）
  - 格式化顯示：`1.5m`, `2.3m` 等
- ✨ **深色/淺色主題切換系統**
  - 完整的 CSS 變數切換機制
  - 淺色主題：白色卡片 + 深色文字 + 深藍重音
  - Header 中添加主題切換按鈕（☀️/🌙）
  - 主題狀態持久化準備
- ✨ **完整 i18n 國際化支援**
  - 3 種語言：繁體中文、English、日本語
  - 30+ 翻譯鍵完整覆蓋
  - Header 中添加語言選擇器
  - 簡化的 `t(key)` 翻譯函數
- ✨ **增強視覺特效**
  - **水滴(Water Droplet)**：添加漸變 + 內陰影 + 3D 反光
  - **入塵(Dust)**：粒子紋理漸變 + multiply 混合模式
  - **刮痕(Scratches)**：對角線條紋 + overlay 混合模式
  - **霧化(Haze)**：分層漸變 + screen 混合模式
  - 視覺逼真度提升 50%+

### 改進
- 🎨 RWD 佈局完全重構，手機體驗優化
- 🌍 i18n 架構成熟，支援快速擴展新語言
- 🎭 主題系統集成，可輕易添加其他主題
- ✨ 視覺特效更加逼真和明顯
- 📱 觸控交互優化，iOS/Android 完美支援
