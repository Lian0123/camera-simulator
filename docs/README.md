# Pro Camera Simulator — Advanced

一個功能強大的相機模擬器，提供專業級別的攝影參數控制和即時預覽效果。

## 特點

✨ **完整的相機模擬**
- 光圈、快門、ISO、白平衡等完整控制
- 多種品牌色彩模擬（尼康、佳能、富士、萊卡）
- 感光元件選擇（全幅、APS-C、M4/3、中畫幅）

🎨 **高級視覺效果**
- 暗角、畸變、色散、散景模擬
- 鏡頭劣化效果（入塵、刮痕、霧化）
- 漏光和水印功能
- 動態模糊和構圖輔助線

📊 **精細控制**
- 固定15點曲線編輯
- 27個快門速度預設
- 像素模擬（1MP ~ 45MP）
- 神經網絡消噪效果

🚀 **現代化設計**
- 毛玻璃效果UI
- 流暢的動畫效果
- 響應式設計
- 繁體中文界面

## 使用方法

### 在線使用
直接在瀏覽器中打開 `index.html` 文件即可使用。

### 功能分類
- **拍攝設定**：光圈、快門、ISO、白平衡等基礎參數
- **後置設定**：濾鏡、曲線、漏光、水印、雜訊控制

## 快速開始

1. 打開應用
2. 在左側預覽照片效果
3. 右側調整各項參數
4. 切換"拍攝設定"和"後置設定"標籤
5. 匯出照片或設定

## 技術棧

- React 18（CDN版本）
- Vanilla CSS 動畫
- HTML5 Canvas（圖片處理）

## 匯出功能

### 匯出照片（PNG）
- 依據像素設定計算輸出解析度
- 套用所有濾鏡和效果
- 包含水印

### 匯出設定（JSON）
- 保存所有參數設定
- 可用於後續恢復
