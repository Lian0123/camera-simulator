### 4️⃣ i18n 国际化架构
✅ **已完成**：多语言翻译结构

**支持语言：**
- zh-Hant（繁体中文）—— 当前默认
- en（英文）—— 预留

**实现方式：**
```javascript
const i18nData = {
  "zh-Hant": { /* 中文翻译 */ },
  "en": { /* 英文翻译 */ }
};
const t = (key) => i18nData[currentLang]?.[key] || key;
```

### 5️⃣ 漏光位置改为多选
✅ **已完成**：从单选改为多选模式

**功能升级：**
- 9 个预设位置（上左、上中、上右、中左、中心、中右、下左、下中、下右）
- 支持同时激活多个位置
- 导出 PNG 时完整保留多漏光设置
- UI 使用 tag 多选样式

**状态结构：**
```javascript
lightLeakPositions: [
  {x:0.1, y:0.2, active:false},
  {x:0.5, y:0.1, active:false},
  // ... 9 个位置
]
```
